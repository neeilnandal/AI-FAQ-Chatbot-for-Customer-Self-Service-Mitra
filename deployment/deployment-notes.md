# Deployment Notes

## Deployment Type

This project can be deployed as a static website because the chatbot is loaded through Botpress Webchat.

## Basic Steps

1. Configure chatbot in Botpress Cloud.
2. Confirm chatbot name, theme, and support flow.
3. Add the Botpress webchat script to the website.
4. Add the local `botpress-embed.js` file.
5. Deploy the static website.
6. Test common support queries.
7. Restrict allowed origins in production.
