# Environment Checklist

## Before Commit

- [ ] No private credentials committed
- [ ] No customer chat logs committed
- [ ] No internal support documents committed
- [ ] FAQ samples are anonymized
- [ ] Screenshots do not reveal private information
- [ ] README includes clear project purpose
- [ ] Security notes are documented

## Before Deployment

- [ ] Botpress chatbot is active
- [ ] Webchat loads correctly
- [ ] Open-chat button works
- [ ] FAQ flows are tested
- [ ] Fallback response is tested
- [ ] Human escalation path is tested
- [ ] `allowedOrigins` is restricted
- [ ] HTTPS is enabled
